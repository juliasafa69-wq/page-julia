🚀 Guia Completo: Configurar Google AdSense na Página Julia

📋 Pré-requisitos

✅ Sua página já está preparada!
✓ Espaços para anúncios criados
✓ Design responsivo
✓ Conteúdo original
✓ Estrutura profissional

🏁 Passo 1: Criar Conta no AdSense

1. Acesse: www.google.com/adsense
2. Clique em "Começar"
3. Faça login com sua conta Google
4. Adicione o URL do seu site
5. Escolha país: Brasil
6. Aceite os termos

📄 Passo 2: Conectar Site ao AdSense

AdSense vai gerar um código como este:
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXX" crossorigin="anonymous"></script>
📍 Onde colocar: Entre as tags <head></head> no seu HTML

🎯 Passo 3: Adicionar Anúncios nos Espaços Preparados

📱 Banner Lateral (300x250)

Substitua este conteúdo:
<div class="ad-sample"> 📢 Espaço para Banner<br> 300x250px </div>
Por este código do AdSense:
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-XXXXXXXXXX" data-ad-slot="YYYYYYYYYY" data-ad-format="auto" data-full-width-responsive="true"></ins> <script> (adsbygoogle = window.adsbygoogle || []).push({}); </script>

📺 Banner Horizontal (728x90)

Local: Banner inferior da sidebar
Código: Similar ao anterior, mas com tamanho específico

📰 Banner no Feed (468x60)

Local: Entre os posts no feed principal
Ideal para: Engajamento máximo

⚡ Passo 4: Anúncios Automáticos (Recomendado)

Vantagem: Google coloca anúncios automaticamente nos melhores locais
Como ativar:
  1. No painel do AdSense
  2. Vá em "Anúncios" → "Por site"
  3. Clique no seu site
  4. Ative "Anúncios automáticos"

📊 Tamanhos de Anúncios Mais Lucrativos

300x250 - Retângulo médio (melhor performance)
728x90 - Leaderboard
320x50 - Banner móvel
336x280 - Retângulo grande
468x60 - Banner padrão

🔧 Passo 5: Otimizações Importantes

1. Performance: Use lazy loading para anúncios
2. Mobile: Anúncios responsivos habilitados ✅
3. Posicionamento: Acima da dobra = mais receita
4. Conteúdo: Mantenha posts regulares
5. SEO: Otimize para palavras-chave valiosas
